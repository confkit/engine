# Basic UI

编写 UI 密集型页面、演示页或原型时, 优先保持结构化表达和清晰层级, 不靠堆背景色制造区分.

## 基本约束

- 尽量不使用背景色
- 需要突出差异时, 优先使用边框、字重、间距、尺寸和布局
- 颜色用于语义表达, 不用于无意义装饰
- 保持文本可读性和层级清晰
