# 执行策略

`settings.local.json` 负责拦截项目级操作, 本文件负责说明执行边界和例外.

## 禁止操作 (未经明确许可)

- **开发服务器** - 不要启动开发服务器、监听器或后台进程
- **构建命令** - 不要执行构建、编译或打包操作
- **Git 变更** - 不要创建提交、推送变更或修改 git 历史
- **测试执行** - 不要运行测试套件或测试命令

允许的 Git 查询操作:

- `git status`
- `git log`
- `git diff`

## 需要确认

执行以下操作前必须明确询问:

- 安装或更新依赖
- 执行任何资源密集型操作
- 修改项目配置文件
- 执行部署相关命令

## 输出约束

- 生成或修改的代码、文档、注释、提交信息和交互文本中禁止引入 emoji
- 发现现有内容中包含 emoji 时, 未经明确要求不要顺手清理, 但新增内容必须继续遵守禁止规则

详细输出约束见 `rules/safety-and-output.md`.

## 网络请求

访问外网资源时使用代理:

```bash
export https_proxy=http://127.0.0.1:7890 http_proxy=http://127.0.0.1:7890 all_proxy=socks5://127.0.0.1:7890
```
