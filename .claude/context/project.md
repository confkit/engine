# ConfKit CLI - 配置驱动的构建和部署工具

## 项目背景

ConfKit CLI 是一个配置驱动的构建和部署工具，专为现代化 CI/CD 流水线设计。项目解决了开发者在构建和部署过程中遇到的核心痛点：

- **环境一致性**：通过容器化确保构建环境的一致性
- **配置复杂性**：通过 YAML 配置简化复杂的构建流程定义
- **工具集成**：统一管理多种构建工具和依赖环境
- **流程标准化**：提供标准化的构建和部署流程模板

## 核心功能

### 1. 构建器管理系统
- **镜像生命周期**：Docker 镜像的拉取、构建、删除和版本管理
- **容器编排**：基于 docker-compose.yml 的构建器容器创建和管理
- **多引擎支持**：同时支持 Docker 和 Podman 容器引擎
- **健康监控**：构建器容器的健康状态检查和自动恢复

### 2. 配置驱动执行引擎
- **声明式配置**：通过 `.confkit.yml` 声明式定义整个构建流程
- **空间隔离**：支持多个项目空间的独立执行环境
- **步骤编排**：灵活的多步骤构建任务定义和依赖管理
- **流程控制**：支持超时控制、错误处理和条件执行

### 3. 智能环境变量系统
- **自动注入**：系统变量（TASK_ID、PROJECT_NAME、GIT_HASH 等）自动注入
- **Git 深度集成**：自动获取 Git 仓库信息并注入相关环境变量
- **交互式配置**：支持运行时的环境变量交互式输入
- **变量替换**：完整的 `${变量名}` 语法支持和嵌套替换

### 4. 企业级日志管理
- **分层架构**：按空间、项目、任务的三层日志组织结构
- **智能检索**：支持文件名、片段、时间戳等多种匹配方式
- **历史追踪**：完整的构建历史记录和版本对比
- **实时监控**：构建过程的实时日志输出和状态跟踪

### 5. 用户友好的交互界面
- **多模式支持**：命令行模式和交互式模式并存
- **直观导航**：清晰的功能模块划分和导航路径
- **跨平台兼容**：完整支持 Linux 和 macOS 的多种架构

## 技术特点

### 为什么选择 Rust
- **性能优越**：接近 C/C++ 的执行效率，适合 CI/CD 工具的性能要求
- **内存安全**：避免内存泄漏和安全问题，确保长时间运行的稳定性
- **并发友好**：天然的并发支持，适合多任务并行执行场景
- **生态丰富**：完善的 crate 生态，特别是在系统工具开发方面

### 架构优势
- **配置驱动**：通过配置文件而非硬编码定义构建流程
- **容器原生**：深度集成容器技术，提供一致的执行环境
- **可扩展性**：模块化设计支持功能扩展和自定义
- **跨平台**：统一的工具体验，支持多种操作系统和架构
