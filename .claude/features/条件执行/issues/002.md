# 条件表达式解析引擎

## 议题目标
实现轻量级条件表达式解析和评估引擎，支持基本比较运算符、逻辑运算符和环境变量引用

## 实现要点
- 创建 ConditionEvaluator 结构体和相关模块
- 支持比较运算符: ==, \!=, >, <, >=, <=
- 支持逻辑运算符: &&, ||, \!
- 支持环境变量引用语法 ${VAR_NAME}
- 实现基于递归下降的表达式解析器
- 添加表达式语法验证功能

## 技术细节
在 src/ 目录下创建新的模块 condition/ ，包含以下核心组件：

1. ConditionEvaluator 主结构体：
   ```rust
   pub struct ConditionEvaluator {
       environment: HashMap<String, String>,
   }
   
   impl ConditionEvaluator {
       pub fn new(environment: HashMap<String, String>) -> Self;
       pub fn evaluate(&self, condition: &str) -> Result<bool>;
       fn parse_expression(&self, expr: &str) -> Result<bool>;
       fn substitute_variables(&self, expr: &str) -> Result<String>;
   }
   ```

2. 表达式解析器实现：
   - 词法分析器：识别操作符、操作数、括号等 token
   - 语法分析器：基于递归下降算法解析表达式树
   - 求值器：遍历表达式树计算最终布尔结果

3. 环境变量替换逻辑：
   - 支持 ${VAR_NAME} 语法
   - 变量不存在时的错误处理
   - 嵌套变量引用的支持
