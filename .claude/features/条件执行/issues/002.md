# 条件表达式词法解析器

## 议题目标
实现基于 nom 的条件表达式词法解析器

## 实现要点
- 支持变量解析: ${VAR}
- 支持字符串字面量: 'value', \

## 技术细节
在 `src/core/condition/parser.rs` 中使用 nom 解析器组合子，构建递归下降解析器。先实现 Token 枚举定义所有词法元素，然后实现各个解析函数，最后组合成完整的表达式解析器。

技术要点：
1. 使用 nom 的 tag, take_while, alt 等组合子
2. 实现变量解析：识别 ${VAR} 格式
3. 字面量解析：支持单双引号字符串和数字
4. 运算符识别：支持所有比较和逻辑运算符
5. 空白字符处理和错误恢复
