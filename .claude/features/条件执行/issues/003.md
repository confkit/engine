# 条件表达式语法解析器

## 议题目标
实现将词法Token转换为ConditionExpression AST的语法解析器

## 实现要点
- 实现运算符优先级处理 (!, >, ==, &&, ||)
- 支持括号改变优先级
- 构建抽象语法树 (AST)
- 提供友好的语法错误提示

## 技术细节
扩展 `src/core/condition/parser.rs`，使用 nom 的组合子实现递归下降语法解析。通过 prec_climb 或手动实现运算符优先级解析，将Token序列转换为ConditionExpression AST。

实现策略：
1. 定义运算符优先级表：! > 比较运算符 > && > ||
2. 使用递归下降解析器处理优先级
3. 构建 AST 节点：Variable, Literal, Binary, Unary
4. 错误恢复和友好提示机制
5. 支持复杂嵌套表达式
