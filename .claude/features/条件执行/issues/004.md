# 条件求值引擎核心实现

## 议题目标
实现ConditionEvaluator核心求值引擎

## 实现要点
- 实现 ConditionEvaluator 结构体和 evaluate 方法
- 支持环境变量查找和空值处理
- 实现各种比较运算 (==, !=, >, <, >=, <=)
- 实现逻辑运算 (&&, ||, !)
- 类型转换和比较逻辑 (字符串 vs 数字)

## 技术细节
在 `src/core/condition/evaluator.rs` 中实现 ConditionEvaluator，通过模式匹配递归遍历 AST。环境变量通过 ExecutionContext 获取，实现智能类型转换（字符串优先，必要时转换为数字）。

核心算法：
1. AST 递归遍历和求值
2. 环境变量解析：从 ExecutionContext 获取值
3. 类型智能转换：字符串、数字、布尔值
4. 比较运算：支持同类型和跨类型比较
5. 逻辑运算：短路求值优化
6. 空值安全处理：None、空字符串的特殊逻辑
