# 条件表达式数据结构设计

## 议题目标
扩展配置结构支持条件字段，定义表达式枚举类型

## 实现要点
- 扩展 ConfKitStepConfig 增加 condition: Option<String> 字段
- 定义 ConditionExpression 枚举表示不同条件类型
- 实现序列化/反序列化支持
- 向后兼容现有配置

## 技术细节
在 `src/types/config.rs` 中扩展步骤配置结构，同时新建 `src/core/condition/mod.rs` 模块定义条件表达式枚举。使用 serde 的 Optional 和 Default 特性确保向后兼容性。

具体实现：
1. 修改 ConfKitStepConfig 结构体增加 condition 字段
2. 创建 condition 模块并定义 ConditionExpression 枚举
3. 实现 Variable, Comparison, Logical, Literal 等表达式类型
4. 确保 YAML 解析兼容性
